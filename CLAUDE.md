# Project

This is a web-search-based AI assistant built using:

- Flask
- Streamlit
- SerpAPI
- BeautifulSoup
- Mixtral 8x7B

Current architecture:

User Query
→ SerpAPI Search
→ Web Scraping
→ LLM
→ Response

Goal:

Transform this project into a production-grade RAG application.

Requirements:

1. Convert to true RAG
2. Add vector embeddings
3. Add document chunking
4. Add source citations
5. Add streaming responses
6. Add chat memory
7. Migrate frontend to Next.js
8. Deploy on Vercel

Important:
Do not modify files until a migration plan is approved.
